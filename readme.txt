Ограничения
-----------
Агрегаторы SIM Roulette не работают с протоколом HTTPS. 
Для установки SR-Navigator на домен с HTTPS следует вынести весь SR-Navigator либо только файл link.php на домен 3-го уровня (HTTP).
Для корректной работы SR-Navigator с SR-Nano на последнем должна быть установлена версия микропрошивки не ниже 3.99.

Дополнительные настройки
------------------------
На агрегаторе следует отключить режим PDU (команда Терминала pdu=0&&save)
На модеме(ах) должен быть включен текстовый режим CMGF=1 (команды Терминала modem>send:AT+CMGF=1 и modem>send:AT&W0, команды следует применить к каждому из модемов агрегатора) 

Установка ПО
------------
1. Распаковать на сервере в корень сайта либо в папку с любым именем (например, SR)
2. Запустить index.php из папки
3. Следовать указаниям инсталлятора
...
4. Настройте CRONTAB для запуска файла cron.php каждую минуту или чаще 
Пример записи для запуска каждую минуту: 
* * * * * /usr/bin/php _путь_к_папке_с_SR_navigator/cron.php > _путь_к_папке_с_логами/cron.log
Пример записи для запуска каждые 10 секунд: 
* * * * * /usr/bin/php _путь_к_папке_с_SR_navigator/cron.php > _путь_к_папке_с_логами/cron.log
* * * * * /usr/bin/php _путь_к_папке_с_SR_navigator/cron.php 10 > _путь_к_папке_с_логами/cron.log
* * * * * /usr/bin/php _путь_к_папке_с_SR_navigator/cron.php 20 > _путь_к_папке_с_логами/cron.log
* * * * * /usr/bin/php _путь_к_папке_с_SR_navigator/cron.php 30 > _путь_к_папке_с_логами/cron.log
* * * * * /usr/bin/php _путь_к_папке_с_SR_navigator/cron.php 40 > _путь_к_папке_с_логами/cron.log
* * * * * /usr/bin/php _путь_к_папке_с_SR_navigator/cron.php 50 > _путь_к_папке_с_логами/cron.log

При использовании HTTPS перенесите файл link.php на специально созданный домен 3-го уровня (link.site.ru) работающий по протоколу HTTP
Вручную внесите изменения в URL для удаленного подключения агрегатора (site.ru/sr/link.php?token=1234567890 -> link.site.ru/link.php?token=1234567890)
